**Example: Press Buttons Using Hook Tool**

Initial State:
- Robot: position (0.5, 0.5), arm retracted
- Hook (tool): position (2.0, 0.5), red, movable
- Button (movable): position (1.5, 1.5), initially red
- Target button: position (2.5, 2.0), initially red

Goal: Press both buttons (turn green).

Steps: Navigate to hook, grasp, press movable button, press target button.
